# 🤖 AI Code Reviewer

An AI-powered code review platform that analyzes source code, detects bugs, suggests optimizations, and provides improved solutions using an LLM through the OpenRouter API.

---

## 🚀 Live Demo

🌐 Frontend: https://ai-code-reviewer-lemon-one.vercel.app

---

## ✨ Features

- 🧠 AI-powered code review
- 💻 Supports multiple programming languages
- 🐞 Bug detection
- ⚡ Code optimization suggestions
- ✅ Improved code generation
- 🎨 Monaco Code Editor
- 📝 Markdown formatted AI responses
- 🌈 Syntax highlighting
- 📱 Responsive UI
- ☁️ Cloud deployment

```bash
npm run dev
```

